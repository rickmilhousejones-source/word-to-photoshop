# Changelog

## 1.25.0 (2026-05-06)

- **CEP 面板**：页码行增加「上一页」「下一页」，与下拉页码联动，首尾页自动禁用对应按钮。
- **CEP 面板**：台词列表支持「回顶」按钮（滚动超过约 80px 后显示）；换页或重绘列表后滚动位置回到顶部。
- **设置**：新增 `bubblePlacementUseParagraphText`（默认 `true`）。在 ScriptUI 设置面板可切换「段落文字 / 点文字」投放对白；`false` 时 CEP 按气泡框对齐策略使用单行点文字（与「仅光标」投放一致），长对白可能溢出框外。
- **迁移**：无需手动操作；未写入 `settings.json` 时按默认使用段落文字。
