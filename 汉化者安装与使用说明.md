# 漫画汉化导入助手 — 汉化者安装与使用说明

面向第一次上手、日常使用与升级的简短指南。**仓库版本（CEP 面板标题）当前为 1.1**。

---

## 一、你需要准备什么

| 项目 | 说明 |
|------|------|
| 操作系统 | **Windows**（CEP 导出、鼠标探针等依赖 PowerShell 与本机路径） |
| Photoshop | 建议 **CC 2018（内部版本约 19.x）及以上**；扩展 manifest 已按此范围配置 |
| Word 稿件 | **`.docx`**（推荐）；扩展名为 `.doc` 但实为 OOXML 的可复制为临时 `.docx` 再读。**不支持 Word 97–2003 真二进制 `.doc`** |
| 本仓库 | 克隆或解压到固定目录（路径尽量**不要频繁挪动**，详见下文「仓库路径」） |

## 二、第一次安装

### 1. 放置仓库

将整个 `word-to-photoshop` 文件夹放到你希望长期使用的路径（例如 `D:\tools\word-to-photoshop`）。

### 2. 安装 CEP 扩展（必做）

**完全重装 CEP**：在资源管理器中进入仓库根目录，双击 **`一键安装-重装插件.cmd`**（名称含义为「一键安装 / 重装」；Windows 文件名不能含 `/`，故用连字符）。脚本会先 **结束 Photoshop 相关进程**，再 **卸载 → 安装** CEP，并尝试启动 Photoshop；首屏有一次 **pause** 供你确认后再执行。  
**仅安装/更新（不杀进程）**：请仍 **右键** `install_cep.ps1` → **使用 PowerShell 运行**，或按下文 PowerShell 命令执行。  
**说明**：该 `.cmd` 提示为 **英文**（避免 `cmd.exe` 默认代码页下 **UTF-8 中文** 解析异常）。

也可 **右键** `install_cep.ps1` → **使用 PowerShell 运行**；或在 PowerShell 里执行：

```powershell
cd "你的仓库路径"
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass -Force
.\install_cep.ps1
```

> 原根目录 **`安装脚本.cmd`** 已废弃。原 **`重装测试.cmd`** / **`reinstall_test.cmd`** 已删除；**完全重装** 逻辑仅在 **`一键安装-重装插件.cmd`**；仅更新扩展请直接运行 **`install_cep.ps1`**。

脚本会：

- 把扩展复制到 **`%APPDATA%\Adobe\CEP\extensions\com.word_to_photoshop.panel`**（若权限允许，也会尝试系统 Common Files 目录）
- 写入 **`host\repo_path.txt`**（记录仓库路径）与用户环境变量 **`WORD_IMPORT_REPO_PATH`**（便于 Photoshop 找到脚本与 `export_docx_styles.ps1`）
- 为 **`CSXS.7`～`CSXS.20`** 注册 **`PlayerDebugMode=1`**（允许加载**未签名**扩展；否则菜单里可能完全不显示）

若脚本提示某个系统目录无法写入，通常不影响：**只要用户目录那一份复制成功即可**。

### 3. 重启 Photoshop

完全退出后再打开。

### 4. 在 Photoshop 里打开面板

- **CC 2018**：菜单多为 **「窗口 → 扩展功能」**，找到 **Word Import CEP**。
- **较新版本**：多为 **「窗口 → 扩展功能（旧版）」→ Word Import CEP**。

若列表里没有该项：

1. 确认已运行 **`install_cep.ps1`**，且存在文件夹  
   `%APPDATA%\Adobe\CEP\extensions\com.word_to_photoshop.panel\CSXS\manifest.xml`
2. 用注册表编辑器查看  
   `HKEY_CURRENT_USER\Software\Adobe\CSXS.8`（或与你 PS 版本对应的 `CSXS.x`）是否存在 **`PlayerDebugMode`**，值为字符串 **`1`**。
3. 仍不行时，可把仓库路径发到群内求助（附上 PS「关于」里的版本号）。

### 5. （可选）完整导入面板（ScriptUI）

CEP 侧按钮 **「打开/关闭完整导入面板」** 会加载仓库里的 **`import_panel.jsx`**（依赖同目录下的 **`import_to_photoshop.jsx`**）。  
也可在 PS 中 **文件 → 脚本 → 浏览**，手动选择 **`import_panel.jsx`**。

首次使用前请确认：**仓库根目录**中有 **`export_docx_styles.ps1`**，且 CEP 能通过 **`repo_path.txt` / `WORD_IMPORT_REPO_PATH`** 找到该目录。

### 6. 字体（可选，在完整导入面板）

在 **完整导入面板（ScriptUI）** 的字体区域可使用：

- **复制字体到项目 fonts…**：选择 `.ttf` / `.otf` / `.ttc`，复制到仓库 **`fonts/`**（同名已存在时会询问是否覆盖）。
- **安装字体到当前用户…**：将字体安装到当前 Windows 用户（用户字体目录 + 注册表项）。

成功后脚本会根据 Photoshop 已安装的字体推测 **常规 / 粗体 PostScript 名**，写回 **`settings.json`**，并刷新面板内的字体下拉。**若下拉里没有新字体**，请 **完全退出 Photoshop 再打开**，然后点 **「刷新系统字体列表」**。  
CEP 快捷面板不提供字体下拉；需要改字体时请用 **「打开/关闭完整导入面板」**。

## 三、第一次使用（推荐流程）

1. **翻译稿 → `.jsxdata`**  
   - **方式 A（CEP）**：在 PS 中打开 **Word Import CEP**，点 **「将翻译稿（docx）转换为导入数据」**，按提示选择 `.docx`/误标 OOXML 的 `.doc`。生成文件默认与 Word **同目录**，扩展名为 **`.jsxdata`**，文件名会与稿件对应（含中文名）。  
   - **方式 B（命令行）**：在仓库根目录打开 PowerShell：  
     `powershell -ExecutionPolicy Bypass -File .\export_docx_styles.ps1 -DocxPath "稿件路径.docx" -OutFile "输出路径.jsxdata" -Minify`

2. **Word 分页标记（需在导出前写在稿件里）**  
   单独成段的 **`#001`、`P01`、`001`** 等形式可作为分页标记（详见脚本注释与仓库内说明）。

3. **PSD + 绑定数据**  
   用 **完整导入面板** 绑定刚生成的 **`.jsxdata`**，选择页码，设置字体与排版参数。

4. **导入文字**  
   使用面板上的 **导入当前页 / 导入全部页**。

5. **CEP 快捷投放台词（可选）**  
   在 CEP 中刷新台词列表，选中一句后在画布上使用点击投放（依赖鼠标探针脚本；若异常可先重启 PS 再试）。

## 四、日常使用贴士

- **尽量不要整条仓库挪盘符**：移动后请重新运行 **`install_cep.ps1`**，以更新 **`repo_path.txt`** 与 **`WORD_IMPORT_REPO_PATH`**。
- **设置**：排版默认值可保存在仓库根目录的 **`settings.json`**（由面板保存写入）。
- **含图 Word**：导出时会跳过正文里的嵌入图节点，**不会因插图整页报错**；图不会在 PSD 里自动生成，仅占位逻辑由脚本侧处理。

## 五、如何更新

### 常规升级步骤

1. **用 Git**：在仓库根目录执行 `git pull`（或下载新压缩包覆盖**除个人设置外**的文件）。
2. **务必再运行一次** **`install_cep.ps1`**：覆盖 CEP 扩展目录中的文件并刷新 **`repo_path.txt`**。
3. **重启 Photoshop**。
4. 若曾改过 **`settings.json`**，一般可保留；合并冲突时以备份为准。

### CEP 面板「检查更新（测试中）」

在 Photoshop 中打开 **Word Import CEP**，在 **常用** 区域点击 **「检查更新（测试中）」**：当前**仅**在浏览器中打开 **腾讯文档** 说明页（不拉取 **`release-channel.json`**）。链接在扩展源码 **`client/main.js`** 的 **`UPDATE_DOCS_URL`** 中维护。

卸载扩展可运行 **`uninstall_cep.ps1`**（会删除扩展目录与用户环境变量 **`WORD_IMPORT_REPO_PATH`**，请按需备份）。
